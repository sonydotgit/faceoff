# TremOff

A simple shell script to
download and trim songs.

## Requires

+ ffmpeg
+ yt-dlp

## Usage

+ Enter youtube link for the song
  and duration into **links.txt**. <br/>
  ytube_link hh:mm:ss hh:mm:ss
+ Run **domp3.sh**
  to download songs.
+ Run **trim.sh**
  present in *down* directory
  to trim the downloaded songs.
